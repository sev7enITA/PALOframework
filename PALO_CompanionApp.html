Eight working tools

From initial screen to evidence package

The Toolbox groups eight focused activities into one local mobile workflow. Each supports preliminary governance analysis and keeps the human decision owner in control.

FRIA Preparation

Operational contribution

The Toolbox across five phases

The mobile app supports evidence activity at each lifecycle decision. It does not replace accountable owners, review gates, source verification, or organization-specific controls.

  1. 01

    Ideation & Screening

    Capture purpose, stakeholders, red flags, and an initial risk tier.

  2. 02

    Assessment & Planning

    Prepare FRIA questions, control checklists, and evidence needs.

  3. 03

    Dev & Validation

    Record test evidence, audit findings, mitigations, and KPIs.

  4. 04

    Deployment

    Maintain monitoring indicators, review notes, and local dossiers.

  5. 05

    Decommissioning

    Document retirement decisions, residual risks, and data disposal.

Trust boundary

Private by architecture

The Toolbox is designed for offline use with no analytics, trackers, or external API calls. Sensitive assessments and corporate documentation remain on the device.

Local evidence handling

The application requests no internet permission for its assessment workflow.

  • Encrypted local SQLite storage with SQLCipher
  • Device biometric protection where available
  • Local PDF and Markdown generation
