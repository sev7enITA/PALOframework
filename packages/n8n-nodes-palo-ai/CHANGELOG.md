# Changelog

## 0.2.0 — Unreleased full-cycle developer preview

- Add the PALO Governed Action node with Verified, Review Required, Denied and Execution Failed outputs.
- Add Action Claim 1.2 and Effect Contract construction for preconditions, expected effects and forbidden effects.
- Route protected execution through the PALO Gateway instead of exposing a direct tool-call output.
- Surface trusted Execution Receipt, Outcome Attestation and Assurance Incident data.
- Keep the 0.1 decision-only PALO Governance node for migration and architecture comparisons.

This version remains unpublished, unaudited and unsuitable for production authorization or consequential tools.

## 0.1.0 — Unreleased developer preview

- Add the installable PALO Governance community-node package scaffold.
- Add encrypted PALO API credentials.
- Add normalized Action Claim generation and immutable approval resume.
- Add explicit Allowed, Approval Required and Denied outputs.
- Add fail-closed response validation, build/lint checks and utility tests.

This version is not published to npm and is not production-ready or n8n-verified.
